# CV de Base - Exemple

Remplacez ce fichier par votre CV en format LaTeX (.tex) ou Markdown (.md).

## Format LaTeX (recommandé)
```latex
\documentclass[11pt,a4paper]{article}
\usepackage[utf8]{inputenc}
\usepackage[french]{babel}

\begin{document}

\section*{Prénom NOM}
\textbf{Email:} votre.email@example.com \\
\textbf{Téléphone:} +33 X XX XX XX XX \\
\textbf{LinkedIn:} linkedin.com/in/votre-profil

\section*{Formation}
\textbf{2024-2026} - Master Data Science, Université XYZ

\section*{Compétences}
Python, SQL, Machine Learning, TensorFlow, AWS

\section*{Expérience}
\textbf{2023-2024} - Stage Data Analyst, Entreprise ABC

\end{document}
```

## Format Markdown (alternatif)
```markdown
# Prénom NOM

**Email:** votre.email@example.com  
**Téléphone:** +33 X XX XX XX XX  
**LinkedIn:** linkedin.com/in/votre-profil

## Formation
- **2024-2026** - Master Data Science, Université XYZ

## Compétences
Python, SQL, Machine Learning, TensorFlow, AWS

## Expérience
- **2023-2024** - Stage Data Analyst, Entreprise ABC
```

L'IA adaptera automatiquement ce CV pour chaque offre d'emploi.
